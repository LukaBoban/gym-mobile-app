### Option 1 ‚Äî Run in **Expo Snack (Recommended)**

1. Open the project in **Expo Snack**
2. Click **Run**
3. Scan the QR code with your phone (Expo Go app)

---

### Option 2 ‚Äî Run Locally with Node.js

1. Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/gym-tracker.git
cd gym-tracker
```

2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npx expo start
```

4. Open on your phone using **Expo Go** or run in a simulator.
